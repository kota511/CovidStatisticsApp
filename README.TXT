PROJECT TITLE: Covid Statistics Explorer (Nickname CovidStats)
PURPOSE OF PROJECT: This project aims to provide a way to navigate through various pages of statistics for COVID-19 on various dates for all London boroughs.
HOW TO START THIS PROJECT: To run this, change the module path in launch.json to the path to your Javafx library. Run the start method in the app.java file, running the main method in the app.java file.
